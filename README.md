**Server:** Node


## Run Locally

Clone the project

```bash
  git clone https://github.com/Mehmet1414/3D-Portfolio
```

Go to the project directory

```bash
  cd 3D-Portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```
